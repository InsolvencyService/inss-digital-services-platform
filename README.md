# inss-digital-services-platform
This repository contains the code for the INSS Digital Services Platform, a comprehensive solution for managing and delivering digital services.

## Features
- User authentication and authorization
- Form management
- Form submission and tracking
- Form configuration
- Form UIs for launching and running forms
- API's for data access, form management and submission
- Unit and integration tests
- Support for multiple databases
